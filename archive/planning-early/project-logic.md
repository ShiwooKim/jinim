# 지님(Jinim) 프로젝트 진행 순서

## 1단계. 브랜드 기준 고정
- 브랜드 한 문장 정의
- 핵심 메시지 3~5개 확정
- 톤앤매너 기준 정리
- 서비스가 하지 않을 것 정의

## 2단계. MVP 범위 확정
- 첫 사용자가 해야 할 행동 정의
- “나의 지님 카드” 정보 구조 설계
- 지님 등록 질문 세트 확정
- 소품함/공개 페이지 범위 결정

## 3단계. 첫 화면/랜딩 구성
- 서비스 소개 문구
- 첫 CTA
- 사용 예시
- 지님 카드 샘플
- 대기자/피드백 수집 구조

## 4단계. 프로토타입 제작
- Figma 또는 바로 웹 MVP
- 모바일 기준 화면 설계
- 등록 플로우 구현
- 카드 생성 결과 화면 구현

## 5단계. 초기 검증
- 내 컬렉션으로 샘플 데이터 제작
- 지인 5~10명에게 테스트
- “등록하고 싶은가 / 공유하고 싶은가 / 다시 보고 싶은가” 검증
- 반응 좋은 질문과 카드 포맷 선별

## 6단계. 공개 MVP
- 도메인 연결
- 기본 SEO/공유 이미지 설정
- 공개 소품함 또는 카드 공유 기능 오픈
- 초기 사용자 피드백 수집
