# 지님(Jinim) 요약 사업계획서

## 1. 서비스 개요

**지님(Jinim)**은 AI가 무엇이든 만들어내는 시대에, 사람이 실제로 지니고 살아온 물건의 의미와 시간을 기록하는 **취향 아카이브 서비스**입니다.

사용자는 시계, 지갑, 펜, 가방, 안경, 카메라처럼 일상 속에서 오래 지녀온 물건을 등록하고, 그 물건에 담긴 선택의 이유와 기억, 취향, 정체성을 기록합니다.

지님은 단순한 소장품 관리 서비스가 아닙니다.  
물건을 통해 사람의 시간과 태도를 기록하는 공간입니다.

> 핵심 문장: 내가 살아온 시간은 여전히 내 것이다.  
> 핵심 질문: 당신에게 있어 지님의 가치란 무엇입니까?  
> 슬로건 후보: 생성된 것보다 지녀온 것. / Find my Time.

---

## 2. 문제 정의

AI 시대가 본격화되면서 글, 이미지, 코드, 기획, 창작물 등 과거에는 사람의 시간과 노력이 필요했던 결과물들이 빠르게 생성되고 복제되고 있습니다.

이 변화는 편리함을 주지만, 동시에 사람들이 전통적으로 믿어온 가치에도 불안을 만듭니다.

- 내가 쌓아온 지식은 앞으로도 가치 있을까?
- 내가 만든 결과물은 여전히 나의 것일까?
- 나의 취향과 정체성은 무엇으로 남을까?
- 생성 가능한 것들이 넘쳐나는 시대에, 진짜 내 것은 무엇일까?

이런 시대일수록 사람들은 다시 유형의 물건에 의미를 부여하게 됩니다.

오래 찬 시계, 매일 쓰는 지갑, 첫 월급으로 산 펜, 늘 들고 다니는 가방처럼 실제로 지니고 살아온 물건에는 시장 가격과 다른 개인의 시간이 담겨 있습니다.

하지만 현재의 커뮤니티와 플랫폼은 대체로 브랜드, 가격, 희소성, 구매 인증, 시세 중심으로 운영됩니다.  
그 결과 물건에 담긴 개인의 이야기와 의미는 충분히 기록되지 못합니다.

---

## 3. 서비스 미션

지님의 미션은 다음과 같습니다.

> 사람이 오래 지닌 물건에 담긴 시간과 의미를 기록하고, 각자가 자기 삶의 가치를 다시 발견하도록 돕는다.

지님은 더 많이 사게 만드는 서비스가 아니라,  
내가 이미 지니고 있거나 앞으로 오래 지니고 싶은 물건의 의미를 발견하게 하는 서비스입니다.

---

## 4. 핵심 타깃

### 초기 타깃

- 시계, 펜, 지갑, 가방 등 일상 소품에 애착을 가진 사람
- 고가 브랜드 과시보다 자기만의 선택 기준과 취향을 중요하게 여기는 사람
- 자신이 가진 물건들을 정리하고, 그 의미를 기록하고 싶은 사람
- 기존 커뮤니티의 과시성, 가격 중심 문화에 피로감을 느끼는 사람
- AI 시대의 변화 속에서 자기 취향과 시간을 다시 확인하고 싶은 사람

### 대표 페르소나

> 나는 비싼 물건을 많이 갖고 싶은 것이 아니다.  
> 나답게 오래 지닐 수 있는 물건을 잘 고르고 싶다.

---

## 5. 핵심 가치

| 가치 | 설명 |
|---|---|
| 소유보다 의미 | 무엇을 가졌는가보다 왜 지니는가에 집중 |
| 자랑보다 이야기 | 구매 인증보다 물건에 담긴 개인의 시간을 기록 |
| 구매보다 발견 | 더 많이 사는 것이 아니라 나다운 물건을 발견 |
| 컬렉션보다 태도 | 수량이나 가격보다 선택 기준과 삶의 방식에 주목 |
| 생성보다 지녀온 시간 | AI가 생성할 수 없는 실제 경험과 흔적을 중시 |

---

## 6. MVP 범위

초기 MVP는 커뮤니티 전체를 만드는 것이 아니라, 사용자가 자신의 물건을 기록하고 공유할 수 있는 **“나의 지님 카드”**를 중심으로 구성합니다.

| 기능 | 설명 |
|---|---|
| 지님 등록 | 사진, 이름, 카테고리, 사용 기간 등 기본 정보 입력 |
| 지님 질문 | 물건을 지니게 된 이유, 함께한 순간, 나에게 주는 의미를 질문 |
| 지님 카드 생성 | 답변을 바탕으로 공유 가능한 카드 형태로 정리 |
| 소품함 | 내가 지닌 물건들을 모아보는 개인 아카이브 |
| 취향 태그 | 단정함, 실용성, 빈티지, 전투용, 기념품, 평생템 등 태그 부여 |
| 컬렉션 분석 | 내가 지닌 물건들의 공통된 취향과 태도를 분석 |
| 다음 지님 추천 | 현재 소품함과 취향을 바탕으로 다음에 어울릴 물건 제안 |
| 공개 소품함 | 사용자가 원할 경우 자신의 지님들을 공개 페이지로 공유 |

---

## 7. 차별화 포인트

기존 서비스가 주로 가격, 브랜드, 모델, 시세, 거래, 착용샷에 집중한다면 지님은 물건과 사람 사이의 관계에 집중합니다.

| 기존 서비스 | 지님 |
|---|---|
| 이 물건은 얼마인가 | 이 물건은 나에게 어떤 의미인가 |
| 어떤 브랜드인가 | 왜 이 물건을 선택했는가 |
| 얼마나 희귀한가 | 얼마나 오래 함께했는가 |
| 지금 시세가 어떤가 | 어떤 시간을 담고 있는가 |
| 남들이 부러워하는가 | 나다운 선택인가 |
| 새로 샀는가 | 앞으로도 계속 지니고 싶은가 |

지님은 시계 애호가 서비스에서 출발할 수 있지만, 최종적으로는 “나다운 물건을 기록하고 발견하는 개인 취향 플랫폼”을 지향합니다.

---

## 8. AI 활용 방향

지님은 AI 시대의 문제의식에서 출발하지만, AI가 사용자의 이야기를 대신 만들어주는 서비스가 되어서는 안 됩니다.

AI는 사용자가 이미 가진 이야기를 더 잘 꺼내고 정리하도록 돕는 역할을 합니다.

| AI 역할 | 설명 |
|---|---|
| 질문자 | 사용자의 기억과 선택 이유를 끌어내는 질문 제공 |
| 편집자 | 사용자의 답변을 담백하고 보기 좋게 정리 |
| 큐레이터 | 여러 지님을 바탕으로 취향의 패턴을 분석 |
| 추천자 | 현재 취향에 어울리는 다음 지님 후보 제안 |
| 아카이비스트 | 시간이 지나며 쌓인 기록을 정리하고 다시 보여줌 |

핵심 원칙은 다음과 같습니다.

> AI가 이야기를 만들어주는 것이 아니라,  
> 사용자가 이미 가진 이야기를 꺼내도록 돕는다.

---

## 9. 커뮤니티 운영 원칙

지님은 사치품 자랑 커뮤니티로 변질되지 않기 위해 초기부터 운영 철학을 명확히 설정합니다.

| 원칙 | 설명 |
|---|---|
| 가격보다 이유 | 물건의 가격보다 사용자가 지니는 이유를 우선 |
| 브랜드보다 이야기 | 브랜드명보다 개인의 기억과 맥락을 중심에 둠 |
| 인기순보다 질문 중심 | 좋아요 경쟁보다 질문별·태그별 탐색 구조 지향 |
| 새것보다 오래 지닌 것 | 구매 인증보다 함께한 시간과 사용 흔적 강조 |
| 고가품보다 나다운 물건 | 저가, 빈티지, 무명 브랜드, 생활 소품도 동등하게 존중 |

## 10. 수익화 가능성

초기에는 사용자의 기록 경험과 공유 경험을 만드는 데 집중하고, 이후 취향 분석과 아카이브 기능을 기반으로 수익화를 검토합니다.

| 수익 모델 | 설명 |
|---|---|
| 프리미엄 소품함 | 더 많은 카드, 커스텀 공개 페이지, 백업, 히스토리 기능 제공 |
| AI 취향 분석 리포트 | 사용자의 지님을 분석해 취향 유형, 추천 방향, 컬렉션 밸런스 제공 |
| 컬렉션북/PDF 출력 | 나의 지님을 하나의 기록물로 만들어 소장 |
| 큐레이션 제휴 | 지님의 철학과 맞는 브랜드, 편집샵, 중고 플랫폼과 연결 |
| 브랜드 협업 콘텐츠 | 과시형 광고가 아닌 이야기 중심의 브랜드 협업 |

수익화 기준은 명확해야 합니다.

> 지님은 더 많이 사게 하는 서비스가 아니라,  
> 더 나다운 것을 오래 지니게 하는 서비스여야 합니다.

## 11. 기대 효과

### 개인 프로젝트 관점

- 창업 전 단계에서 작게 검증 가능한 서비스
- 기획, 브랜딩, AI 큐레이션, 커뮤니티 설계 역량을 모두 보여줄 수 있음
- 본인이 직접 첫 번째 핵심 유저가 될 수 있는 높은 몰입도
- 작은 MVP로도 완성도 있는 결과물을 만들 수 있음

### 실제 서비스 관점

- 기존 시계 커뮤니티/컬렉션 앱과 다른 철학적·감성적 포지션 확보 가능
- 시계에서 시작해 일상 소품 전반으로 확장 가능한 구조
- AI 시대에 더 중요해지는 “개인의 실제 경험과 시간”을 기록하는 서비스로 차별화 가능
- 과시성 소비가 아닌 “나다운 선택”을 원하는 니치 유저층 공략 가능
