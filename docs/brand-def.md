# 지님(Jinim) 브랜드 기준 고정 문서

## 1. 문서 목적

이 문서는 지님(Jinim)의 브랜드 방향과 제품 판단 기준을 고정하기 위한 기준 문서다.

지님은 단순한 소장품 관리 앱이나 시계 커뮤니티가 아니다.  
AI가 무엇이든 만들어내는 시대에, 사람이 실제로 지니고 살아온 물건의 의미와 시간을 기록하는 취향 아카이브 서비스다.

앞으로 기능, 화면, 카피, AI 활용, 커뮤니티 운영을 결정할 때 이 문서를 기준으로 삼는다.

---

## 2. 브랜드 한 문장 정의

**지님은 AI가 무엇이든 만들어내는 시대에, 사람이 실제로 지니고 살아온 물건의 의미와 시간을 기록하는 취향 아카이브 서비스입니다.**

조금 더 제품에 가까운 표현은 다음과 같다.

**지님은 내가 오래 지닌 물건을 통해 나의 시간, 취향, 정체성을 기록하는 서비스입니다.**

---

## 3. 브랜드가 출발한 문제의식

AI 시대가 본격화되면서 글, 이미지, 코드, 기획, 창작물 등 많은 결과물이 빠르게 생성되고 복제되고 있다.

이 변화는 편리하지만, 동시에 사람들에게 다음과 같은 불안을 만든다.

- 내가 쌓아온 지식은 앞으로도 가치 있을까?
- 내가 만든 결과물은 여전히 나의 것일까?
- 나의 취향과 정체성은 무엇으로 남을까?
- 생성 가능한 것들이 넘쳐나는 시대에, 진짜 내 것은 무엇일까?

지님은 이 질문에 대해 “실제로 지니고 살아온 물건”이라는 방향에서 답을 찾는다.

오래 찬 시계, 매일 쓰는 지갑, 첫 월급으로 산 펜, 늘 들고 다니는 가방처럼 실제로 지니고 살아온 물건에는 시장 가격과 다른 개인의 시간이 담겨 있다.

**내가 살아온 시간은 여전히 내 것이다.**

지님은 이 감각을 제품으로 만든다.

---

## 4. 브랜드 미션

**사람이 오래 지닌 물건에 담긴 시간과 의미를 기록하고, 각자가 자기 삶의 가치를 다시 발견하도록 돕는다.**

지님은 더 많이 사게 만드는 서비스가 아니다.  
내가 이미 지니고 있거나, 앞으로 오래 지니고 싶은 물건의 의미를 발견하게 하는 서비스다.

---

## 5. 브랜드 비전

지님은 장기적으로 **개인의 취향과 시간을 기록하는 아카이브 플랫폼**을 지향한다.

초기에는 시계에서 시작할 수 있다.  
시계는 시간의 상징이자, 몸에 지니는 물건이며, 취향과 태도를 잘 드러내는 카테고리이기 때문이다.

하지만 지님의 본질은 시계가 아니다.  
지님이 다루는 본질은 **사람이 오래 지니고 싶은 물건에 담긴 의미**다.

따라서 지님은 시계에서 출발하더라도, 지갑, 펜, 가방, 안경, 카메라, 키링, 노트 등 일상 속 다양한 물건으로 확장될 수 있다.

---

## 6. 핵심 질문

지님이 사용자에게 던지는 가장 중요한 질문은 다음과 같다.

> **당신에게 있어 지님의 가치란 무엇입니까?**

이 질문은 물건에 대해 묻는 것처럼 보이지만, 실제로는 사람에 대해 묻는다.

- 왜 이 물건을 선택했는가?
- 왜 아직도 지니고 있는가?
- 왜 비싸지 않아도 소중한가?
- 왜 이것이 나다운 물건인가?
- 앞으로도 계속 지니고 싶은 이유는 무엇인가?

지님은 물건의 자랑이 아니라, 물건을 통해 드러나는 사람의 이야기를 기록한다.

## 7. 핵심 브랜드 메시지

### 대표 메시지

**내가 살아온 시간은 여전히 내 것이다.**

### 보조 메시지

- 생성된 것보다 지녀온 것.
- 오래 지닐 물건에는 이유가 있습니다.
- 당신이 지닌 물건에는 당신의 시간이 담겨 있습니다.
- 지님은 물건을 통해 사람의 시간과 태도를 기록하는 공간입니다.
- AI가 무엇이든 만들어내는 시대에도, 당신이 지녀온 시간은 대체되지 않습니다.

### 영문 메시지 후보

- Find my Time.
- What you carry, carries you.
- Things that hold your time.
- Your time, held in things.

## 8. 브랜드 가치

지님은 시장 가격이 아니라 개인의 시간을 기준으로 물건의 가치를 바라본다.

| 가치 | 의미 |
|---|---|
| 소유보다 의미 | 무엇을 가졌는가보다 왜 지니는가에 집중한다. |
| 자랑보다 이야기 | 구매 인증보다 물건에 담긴 개인의 시간을 기록한다. |
| 구매보다 발견 | 더 많이 사는 것이 아니라 나다운 물건을 발견한다. |
| 컬렉션보다 태도 | 수량이나 가격보다 선택 기준과 삶의 방식을 본다. |
| 생성보다 지녀온 시간 | AI가 생성할 수 없는 실제 경험과 흔적을 중시한다. |

## 9. 지님이 바라보는 가치의 기준

기존 시장은 물건의 가치를 주로 가격, 브랜드, 희소성, 시세, 상태로 판단한다.  
지님은 다음 기준으로 물건의 가치를 바라본다.

| 가치 기준 | 설명 |
|---|---|
| 시간 가치 | 얼마나 오래 함께했는가 |
| 기억 가치 | 어떤 순간과 연결되어 있는가 |
| 취향 가치 | 나다운 선택인가 |
| 사용 가치 | 실제 내 삶에서 쓰이고 있는가 |
| 태도 가치 | 내가 어떤 삶을 지향하는지 보여주는가 |
| 전승 가치 | 누군가에게 이야기로 남길 수 있는가 |
| 회복 가치 | 불안한 시대에 나를 붙잡아주는가 |

## 10. 포지셔닝

지님은 기존의 소장품 서비스, 시계 커뮤니티, 중고거래 플랫폼과 다르다.

| 기존 서비스의 관점 | 지님의 관점 |
|---|---|
| 이 물건은 얼마인가 | 이 물건은 나에게 어떤 의미인가 |
| 어떤 브랜드인가 | 왜 이 물건을 선택했는가 |
| 얼마나 희귀한가 | 얼마나 오래 함께했는가 |
| 지금 시세가 어떤가 | 어떤 시간을 담고 있는가 |
| 남들이 부러워하는가 | 나다운 선택인가 |
| 새로 샀는가 | 앞으로도 계속 지니고 싶은가 |

지님은 소비를 부추기는 서비스가 아니라, 소유의 의미를 다시 묻는 서비스다.

## 11. 핵심 사용자

지님의 초기 사용자는 물건을 많이 가진 사람이 아니다.  
자신이 지닌 물건의 의미를 생각해본 적 있는 사람이다.

### 초기 타깃

- 시계, 펜, 지갑, 가방 등 일상 소품에 애착을 가진 사람
- 고가 브랜드 과시보다 자기만의 선택 기준을 중요하게 여기는 사람
- 자신의 취향을 정리하고 싶은 사람
- 기존 커뮤니티의 과시성, 가격 중심 문화에 피로감을 느끼는 사람
- AI 시대의 변화 속에서 자기 취향과 시간을 다시 확인하고 싶은 사람

### 대표 페르소나

> 나는 비싼 물건을 많이 갖고 싶은 것이 아니다.  
> 나답게 오래 지닐 수 있는 물건을 잘 고르고 싶다.

---

## 12. 제품 경험 원칙

지님의 첫 경험은 무겁지 않아야 한다.

사용자는 처음부터 “AI 시대의 정체성 회복”을 하러 오지 않을 수 있다.  
더 현실적인 욕구에서 시작할 가능성이 높다.

- 내 물건을 예쁘게 정리하고 싶다.
- 내가 가진 물건에 설명을 붙이고 싶다.
- 내 취향을 분석받고 싶다.
- 공유할 만한 카드가 있으면 좋겠다.
- 다음에 뭘 사야 할지 고민된다.
- 자랑이 아니라 이야기로 공유하고 싶다.

따라서 제품은 다음 순서로 경험되어야 한다.

> 먼저 “내 물건을 기록해보세요”로 시작하고,  
> 사용자가 기록하는 과정에서 “이건 내 시간을 기록하는 일이구나”를 느끼게 한다.

## 13. MVP 기준

초기 MVP는 커뮤니티 전체가 아니라, 사용자가 자신의 물건을 기록하고 공유할 수 있는 **나의 지님 카드**를 중심으로 구성한다.

### MVP 핵심 기능

| 기능 | 설명 |
|---|---|
| 지님 등록 | 사진, 이름, 카테고리, 사용 기간 등 기본 정보 입력 |
| 지님 질문 | 물건을 지니게 된 이유, 함께한 순간, 나에게 주는 의미를 질문 |
| 지님 카드 생성 | 답변을 바탕으로 공유 가능한 카드 형태로 정리 |
| 소품함 | 내가 지닌 물건들을 모아보는 개인 아카이브 |
| 취향 태그 | 단정함, 실용성, 빈티지, 전투용, 기념품, 평생템 등 태그 부여 |
| 컬렉션 분석 | 내가 지닌 물건들의 공통된 취향과 태도를 분석 |
| 공개 소품함 | 사용자가 원할 경우 자신의 지님들을 공개 페이지로 공유 |

### MVP 이후 검토 기능

| 기능 | 설명 |
|---|---|
| 다음 지님 추천 | 현재 소품함과 취향을 바탕으로 다음에 어울릴 물건 제안 |
| 유사 취향 탐색 | 비슷한 물건과 취향을 가진 사용자 발견 |
| 주제별 지님 모음 | 오래 지닌 물건, 비싸지 않지만 소중한 것 등 테마별 탐색 |
| 컬렉션북/PDF | 나의 지님 기록을 하나의 문서나 이미지로 출력 |

## 14. 첫 경험 플로우

지님의 첫 경험은 다음 흐름을 기준으로 설계한다.

1. 내가 가진 물건 하나를 등록한다.
2. 몇 가지 좋은 질문을 받는다.
3. 답하다 보니 그 물건의 의미를 다시 생각하게 된다.
4. AI가 사용자의 답변을 담백하게 정리해준다.
5. 예쁜 지님 카드가 생성된다.
6. 사용자는 다른 물건도 등록해보고 싶어진다.

이 경험의 목표는 사용자가 브랜드 철학을 읽고 감동하는 것이 아니다.

**기록하다가 자기 이야기를 발견하게 만드는 것.**

## 15. 지님 카드 기준

지님 카드는 단순한 제품 정보 카드가 아니다.  
물건의 스펙보다 사용자와 물건의 관계를 담는 카드다.

### 카드 구성 예시

| 항목 | 설명 |
|---|---|
| 물건 이름 | 사용자가 등록한 물건명 |
| 별명 | 사용자가 부르는 이름 |
| 카테고리 | 시계, 펜, 지갑, 가방 등 |
| 함께한 시간 | 사용 기간 |
| 역할 | 이 물건이 내 삶에서 맡는 역할 |
| 무드 | 이 물건이 가진 분위기 |
| 지닌 이유 | 이 물건을 선택하고 계속 지니는 이유 |
| 함께한 순간 | 이 물건과 연결된 기억 |
| 나에게 주는 감각 | 안정감, 자신감, 실용성, 위로 등 |
| 지님의 가치 | 사용자가 직접 정의한 개인적 가치 |

## 16. 질문 설계 기준

지님은 좋은 질문을 통해 좋은 이야기를 끌어낸다.  
질문은 서비스의 핵심 자산이다.

### 질문 설계 원칙

- 사용자를 평가하지 않는다.
- 구매를 유도하지 않는다.
- 물건의 가격보다 이유를 묻는다.
- 브랜드보다 기억과 맥락을 묻는다.
- 과한 감성 답변을 강요하지 않는다.
- 짧게 답해도 자연스럽게 기록이 완성되어야 한다.

### 등록 질문 예시

| 질문 | 의도 |
|---|---|
| 이 물건을 언제부터 지니고 있나요? | 시간성 |
| 왜 이 물건을 선택했나요? | 선택 기준 |
| 이 물건은 당신에게 어떤 역할인가요? | 정체성 |
| 가장 많이 함께한 순간은 언제인가요? | 기억 |
| 비싸서가 아니라 좋은 이유는 무엇인가요? | 개인 가치 |
| 앞으로도 계속 지닐 것 같나요? | 지속성 |
| 이 물건을 통해 당신에 대해 알 수 있는 것은 무엇인가요? | 자기 이해 |

## 17. AI 활용 기준

지님은 AI 시대의 문제의식에서 출발하지만, AI가 사용자의 이야기를 대신 만들어주는 서비스가 되어서는 안 된다.

AI는 조용한 조력자여야 한다.

### AI가 해야 할 역할

| 역할 | 설명 |
|---|---|
| 질문자 | 사용자가 자신의 기억과 이유를 떠올리도록 돕는다 |
| 편집자 | 사용자의 답변을 담백하게 정리한다 |
| 큐레이터 | 여러 지님을 바탕으로 취향의 패턴을 읽어준다 |
| 추천자 | 현재 소품함과 취향을 바탕으로 다음 지님을 제안한다 |
| 아카이비스트 | 시간이 지나며 쌓인 기록을 정리하고 다시 보여준다 |

### AI가 하지 말아야 할 역할

| 역할 | 이유 |
|---|---|
| 사연 생성기 | 진정성을 훼손한다 |
| 허세 문구 생성기 | 브랜드 톤을 망친다 |
| 구매 유도 봇 | 지님을 소비 서비스로 축소한다 |
| 가격 평가 중심 봇 | 기존 시세 서비스와 겹친다 |
| 과한 감성 편집자 | 사용자의 실제 이야기를 왜곡한다 |

핵심 원칙은 다음과 같다.

> AI가 이야기를 만들어주는 것이 아니라,  
> 사용자가 이미 가진 이야기를 꺼내도록 돕는다.

## 18. 커뮤니티 운영 기준

지님은 커뮤니티로 확장될 수 있지만, 일반적인 피드형 커뮤니티를 그대로 따라가서는 안 된다.

인기순 피드, 좋아요 중심 노출, 고가 브랜드 중심 콘텐츠는 지님을 쉽게 사치품 자랑 커뮤니티로 만들 수 있다.

### 운영 원칙

| 원칙 | 설명 |
|---|---|
| 가격보다 이유 | 물건의 가격보다 사용자가 지니는 이유를 먼저 보여준다 |
| 브랜드보다 이야기 | 브랜드명보다 개인의 기억과 맥락을 중심에 둔다 |
| 인기순보다 질문 중심 | 좋아요 경쟁보다 질문별·태그별 탐색을 우선한다 |
| 새것보다 오래 지닌 것 | 구매 인증보다 함께한 시간과 사용 흔적을 강조한다 |
| 고가품보다 나다운 물건 | 저가, 빈티지, 무명 브랜드, 생활 소품도 동등하게 존중한다 |

지님에서 중요한 것은 “무엇을 가졌는가”가 아니라 “왜 지니고 있는가”다.

## 19. 톤앤매너 기준

지님은 깊은 브랜드 철학을 가지되, 사용자에게는 담백하고 생활감 있게 말해야 한다.

### 지향하는 톤

- 담백한
- 사려 깊은
- 정제된
- 따뜻한
- 조용하지만 선명한
- 취향을 존중하는
- 시간이 느껴지는

### 지양하는 톤

- 과시적인
- 소비를 부추기는
- 지나치게 감성적인
- 허세 섞인
- 브랜드/가격 중심의
- 랭킹과 경쟁을 조장하는

### 톤 원칙

> 브랜드 철학은 깊게.  
> 제품 경험은 가볍게.  
> 커뮤니티는 천천히.  
> 질문은 날카롭게.  
> AI는 조용히.

## 20. 제품 카피 기준

브랜드 철학을 설명할 때는 깊은 문장을 사용할 수 있지만, 실제 제품 화면에서는 더 짧고 담백하게 말한다.

### 브랜드 철학용 문장

- 생성된 것보다 지녀온 것.
- 내가 살아온 시간은 여전히 내 것이다.
- AI가 무엇이든 만들어내는 시대에도, 당신이 지녀온 시간은 대체되지 않습니다.

### 제품 경험용 문장

- 오래 지닐 물건에는 이유가 있습니다.
- 지님은 그 이유를 기록하는 공간입니다.
- 당신이 지닌 물건에는 당신의 시간이 담겨 있습니다.
- 내 물건 하나를 기록해보세요.
- 이 물건을 계속 지니고 싶은 이유는 무엇인가요?

### 커뮤니티용 질문

- 당신에게 있어 지님의 가치란 무엇입니까?
- 비싸서가 아니라 좋은 이유는 무엇인가요?
- 오래 지닌 물건 하나를 소개해주세요.
- 이 물건은 당신에게 어떤 시간을 담고 있나요?

## 21. 지님이 해야 할 것

지님은 다음을 해야 한다.

- 사람이 실제로 지니고 살아온 물건을 기록하게 한다.
- 물건의 가격보다 의미와 이유를 묻게 한다.
- 사용자가 자기 취향을 발견하도록 돕는다.
- 오래 지닌 물건의 가치를 존중한다.
- 저가, 빈티지, 생활 소품도 동등하게 다룬다.
- 사용자의 이야기를 담백하게 정리한다.
- 공유하고 싶은 지님 카드를 만든다.
- AI를 조용한 질문자이자 편집자로 사용한다.
- 커뮤니티가 생기더라도 과시보다 이야기를 중심에 둔다.

## 22. 지님이 하지 말아야 할 것

지님은 다음을 하지 않는다.

- 고가 제품을 더 가치 있게 취급하지 않는다.
- 가격, 시세, 리셀가를 중심 경험으로 만들지 않는다.
- 기추 인증 중심 커뮤니티가 되지 않는다.
- 인기순, 랭킹, 좋아요 경쟁에 의존하지 않는다.
- 사용자가 더 많이 사도록 조급하게 만들지 않는다.
- AI가 없는 사연을 대신 만들어내지 않는다.
- 과한 감성 문장으로 사용자의 이야기를 덮어쓰지 않는다.
- 시계에만 갇히는 브랜드가 되지 않는다.
- 단순한 중고거래나 시세 조회 서비스로 포지셔닝하지 않는다.
- 사치품 자랑 커뮤니티가 되지 않는다.

## 23. 의사결정 기준

앞으로 기능이나 콘텐츠를 추가할 때 다음 질문으로 판단한다.

### 브랜드 적합성 질문

- 이 기능은 물건보다 사람의 이야기를 더 잘 드러내는가?
- 이 기능은 가격보다 이유를 묻게 하는가?
- 이 기능은 사용자가 자기 취향을 발견하도록 돕는가?
- 이 기능은 오래 지닌 물건의 가치를 존중하는가?
- 이 기능은 과시나 비교를 부추기지 않는가?
- 이 기능은 AI가 사용자의 이야기를 대신 만들지 않도록 설계되어 있는가?
- 이 기능은 지님이 시계 밖으로 확장될 수 있는 구조를 해치지 않는가?

하나라도 강하게 어긋난다면, 기능 추가를 보류하거나 다른 방식으로 설계한다.

## 24. 최종 기준 문장

지님은 물건을 자랑하는 서비스가 아니라,  
**물건을 통해 나를 다시 이해하는 서비스**여야 한다.

지님은 더 많이 사게 하는 서비스가 아니라,  
**더 나다운 것을 오래 지니게 하는 서비스**여야 한다.

지님은 이야기를 만들어내는 서비스가 아니라,  
**이미 살아온 시간의 이야기를 꺼내는 서비스**여야 한다.
