# 지님 브랜드웹 0.1 섹션별 비주얼 기획안

## 1. 기획 목적

현재 브랜드웹 0.1은 브랜드 철학과 메시지는 잘 전달되지만, 비주얼 밀도가 낮아 “좋은 선언문”에 더 가깝게 보인다.

이번 비주얼 작업의 목적은 다음과 같다.

- 지님의 브랜드 톤을 시각적으로 강화한다.
- “오래 지닌 물건”의 감각을 보여준다.
- 텍스트 중심 구조에 리듬과 밀도를 더한다.
- 제품이 아직 완성되지 않았더라도, 앞으로의 경험을 상상할 수 있게 만든다.
- 단순히 예쁜 이미지를 넣는 것이 아니라, **지님만의 비주얼 시스템**을 만든다.

---

## 2. 전체 비주얼 방향

### 비주얼 키워드

- Warm Minimal
- Quiet Archive
- Personal Objects
- Time Worn
- Soft Brown
- Subtle Memory
- Editorial but calm

### 표현해야 할 감각

- 오래 지닌 물건
- 손때가 묻은 시간
- 조용한 취향
- 과시적이지 않은 정돈감
- 기록과 아카이브
- 따뜻한 물성
- 시계에 갇히지 않는 확장성

### 피해야 할 방향

- 시계 광고처럼 보이는 연출
- 럭셔리/하이엔드 브랜드 느낌
- 너무 감성적인 손글씨 무드
- 카페/소품샵 같은 브라운 감성
- 이미지마다 톤이 제각각인 구성
- 과도하게 인물 중심인 비주얼

---

## 3. 스타일 기준

### 기본 스타일

**Warm Minimal Still Life + Product Mockup 혼합**

### 이미지 구성 원칙

- 메인 이미지는 **정물(still life)** 중심
- 오브젝트는 3~6개 정도로 과하지 않게
- 배경은 아이보리/웜톤/정제된 표면
- 브라운, 베이지, 차콜 계열 중심
- 그림자는 부드럽고 얕게
- 사용감은 “적당히” 느껴지게
- 브랜드 철학상 과시보다 의미가 느껴지는 구성

### 주요 오브젝트 후보

- 시계
- 지갑
- 펜/만년필
- 안경
- 노트/카드
- 키링
- 작은 카메라
- 태그/라벨
- 카드/보관함

## 4. 섹션별 비주얼 계획

## A. Hero 섹션

### 현재 역할
- 브랜드 첫인상 전달
- “당신에게 있어 지님의 가치란 무엇입니까?”라는 질문 강화

### 필요한 비주얼
- **대표 키비주얼 1장**
- 필요 시 모바일 대응용 크롭 버전 1장

### 이미지 콘셉트
오래 지닌 물건들이 정돈된 테이블 위에 놓여 있는 조용한 정물 이미지.  
시계, 가죽 지갑, 펜, 카드, 안경 같은 오브젝트가 함께 있으며, “개인의 시간”이 느껴지는 구성이어야 한다.

### 목적
- 텍스트에 감정적 밀도 부여
- “지님이 다루는 세계”를 한 장으로 전달
- 시계 서비스가 아니라, 지닌 물건 전반을 다루는 브랜드라는 점을 암시

### 구현 방향
- 데스크탑: 텍스트 옆 또는 아래에 배치
- 모바일: 텍스트 아래 배치
- 너무 크거나 장식적으로 쓰기보다 여백 있는 방식 추천

### 파일명 예시
- `/public/images/hero/jinim-hero-01.png`
- `/public/images/hero/jinim-hero-01-mobile.png`

## B. “무엇이 진짜 나의 것일까요?” 섹션

### 현재 역할
- AI 시대의 문제의식 전달
- 지님의 출발점 설명

### 필요한 비주얼
- **보조 정물 이미지 1장**

### 이미지 콘셉트
오래 사용한 물건 몇 개가 자연스럽게 놓여 있는 장면.  
Hero보다 조금 더 가까이, 더 개인적인 느낌.  
예: 작은 시계, 메모 카드, 펜, 낡은 가죽 소품.

### 목적
- 텍스트의 추상성을 물성으로 보완
- “생성된 것보다 지녀온 것”의 감각 전달

### 구현 방향
- 텍스트 옆 1/2 컬럼
- 또는 텍스트 아래 full-width가 아닌 적당한 폭

### 파일명 예시
- `/public/images/sections/section-meaning-01.png`

## C. “지님이 하는 일” 섹션

### 현재 역할
- 서비스의 정의 설명

### 필요한 비주얼
- **제품형 비주얼 1~2장**
- 지님 카드 mockup 1장
- 소품 등록/기록 감각을 보여주는 이미지 1장

### 이미지 콘셉트
실제 서비스가 앞으로 제공할 경험을 보여주는 장면.  
예:
- 하나의 물건과 그에 대한 기록 카드가 함께 보이는 이미지
- “물건 → 질문 → 기록”의 흐름을 보여주는 mockup
- UI 느낌이 살짝 섞인 product visual

### 목적
- 브랜드 철학이 실제 서비스 경험으로 이어진다는 점 전달
- 사용자가 “내가 들어오면 뭘 하게 되는지” 감을 잡게 함

### 구현 방향
- 카드 2개 정도를 나란히 보여주거나
- 한쪽엔 텍스트, 한쪽엔 sample card mockup

### 파일명 예시
- `/public/images/cards/card-sample-01.png`
- `/public/images/cards/card-sample-02.png`

## D. “무엇이 다른가요?” 섹션

### 현재 역할
- 기존 시선 vs 지님의 시선 비교

### 필요한 비주얼
- **필수 아님**
- 있더라도 아주 가벼운 그래픽 또는 아이콘 수준

### 추천 방향
이 섹션은 지금처럼 표 중심이 더 명확하다.  
이미지를 과하게 넣기보다, 표의 가독성과 분위기를 보완하는 정도가 적절하다.

### 추가 가능 요소
- 행 앞 작은 아이콘
- 표 위/옆에 아주 작은 카드/태그 심볼
- 브랜드 컬러 라인/포인트 그래픽

### 목적
- 시각적 심심함 완화
- 정보 전달은 유지

---

## E. “바라보는 가치” 섹션

### 현재 역할
- 브랜드 가치 정리
- 카드형 정보 전달

### 필요한 비주얼
- **오브젝트 디테일 컷 2~3장** 또는
- 카드형 섹션 배경/작은 이미지

### 이미지 콘셉트
가치 항목과 연결될 수 있는 작은 정물/오브젝트 컷.
예:
- 소유보다 의미 → 물건 하나와 짧은 기록 카드
- 자랑보다 이야기 → 메모 카드와 펜
- 구매보다 발견 → 소품함/정리된 트레이
- 생성보다 지녀온 시간 → 사용감 있는 금속/가죽 디테일

### 목적
- 정보 카드의 추상성을 완화
- 가치를 더 기억에 남게 함

### 구현 방향
- 각 카드에 작은 아이콘처럼 넣거나
- 섹션 하단/상단에 2~3개 이미지 블록 배치

### 파일명 예시
- `/public/images/sections/value-01.png`
- `/public/images/sections/value-02.png`
- `/public/images/sections/value-03.png`

## F. “더 묻는 질문들” 섹션

### 현재 역할
- 질문 중심 브랜드 톤 강화

### 필요한 비주얼
- **선택 사항**
- 필수는 아님

### 추천 방향
이 섹션은 지금처럼 질문 텍스트가 주인공인 것이 좋다.  
대신 배경 질감이나 아주 얕은 그래픽 요소 정도만 넣는 게 적절하다.

### 추가 가능 요소
- 종이/카드 텍스처
- 얕은 라인 그래픽
- 작은 인용부호/질문 태그 심볼

### 목적
- 텍스트 몰입도 유지
- 과도한 장식 방지

---

## G. “곧 만나는 경험” 섹션

### 현재 역할
- 앞으로의 서비스 경험 예고

### 필요한 비주얼
- **제품 UI mockup 1~2장**
- 공개 소품함 예시 1장
- 지님 카드 묶음 1장

### 이미지 콘셉트
실제 MVP를 상상할 수 있는 구체적인 제품 비주얼.
예:
- “내 소품함” 미리보기
- 지님 카드 리스트
- 물건 등록 후 카드가 만들어진 화면 mock
- 모바일 화면/카드 격자

### 목적
- 브랜드웹이 “제품 없는 철학 페이지”로 보이지 않게 함
- 사용자가 기대할 수 있는 구체적 경험 제시

### 구현 방향
- 2~3개 화면이 나란히 보이는 mockup
- 카드 그리드 형태도 가능

### 파일명 예시
- `/public/images/ui/ui-preview-01.png`
- `/public/images/ui/ui-preview-02.png`
- `/public/images/ui/ui-preview-03.png`

## H. 마지막 브랜드 선언 섹션

### 현재 역할
- “내가 살아온 시간은 여전히 나의 것입니다.”로 브랜드 정리

### 필요한 비주얼
- **필수는 아님**
- 아주 절제된 보조 텍스처 정도 가능

### 추천 방향
이 섹션은 텍스트 힘이 좋아서 이미지가 없어도 됨.  
오히려 여백을 살리는 편이 더 좋다.

### 추가 가능 요소
- 얕은 배경 그라데이션
- 종이/아이보리 질감
- 아주 작은 심볼

## 5. 이미지 자산 목록 (0.1 기준 최소 세트)

### 필수 세트

| 구분 | 수량 | 우선순위 |
|---|---:|---:|
| Hero 키비주얼 | 2 | 높음 |
| 보조 정물 이미지 | 2 | 높음 |
| 지님 카드 샘플 | 3~4 | 높음 |
| UI mockup | 2~3 | 높음 |
| 가치 섹션용 오브젝트 컷 | 2~3 | 중간 |
| 배경/텍스처 | 1~2 | 중간 |
| OG 이미지 | 1 | 중간 |

### 권장 총량
**12~15개 내외**

너무 많이 만들기보다, 일관성 있는 세트를 먼저 만드는 게 중요하다.
